# Results Summary

## Dataset Statistics

Total Samples: 38,925

Legitimate Samples: 25,809

Phishing Samples: 13,116

## Best Performing Model

Email + URL + Webpage Fusion

## Performance Metrics

| Metric | Value |
|----------|----------|
| Accuracy | 96.3% |
| Precision | 95.8% |
| Recall | 95.1% |
| F1-Score | 95.4% |
| ROC-AUC | 0.987 |

## Key Findings

- Multimodal fusion outperformed all unimodal baselines.
- Email, URL, and webpage information provide complementary phishing indicators.
- Fusion improved robustness against phishing attacks.
- The framework reduced false positives and improved overall detection reliability.
- Explainability improves analyst trust and transparency.

## Research Contributions

- Multimodal phishing detection framework.
- Integration of email, URL, and webpage evidence.
- Explainable phishing intelligence.
- Robust phishing detection evaluation.
- Operational decision support for cybersecurity analysts.
