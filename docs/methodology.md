# Methodology

## Overview

This project presents a Multimodal Phishing Intelligence Framework that combines email content, URL characteristics, and webpage information to improve phishing detection performance.

## Datasets

### Email Modality
- Kaggle Phishing Email Dataset

### URL Modality
- PhiUSIIL Phishing URL Dataset

### Webpage Modality
- Mendeley Phishing Website Dataset

## Workflow

1. Data Collection
2. Data Preprocessing
3. Email Feature Extraction
4. URL Feature Extraction
5. Webpage Feature Extraction
6. Unimodal Model Training
7. Bimodal Fusion
8. Trimodal Fusion
9. Evaluation
10. Result Analysis

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC
- Brier Score
- Expected Calibration Error (ECE)
